How it works

Three rings of defense

Most tools read config files. ClawArmor also probes your live running gateway — so you know you're actually safe, not just configured correctly.

RING 1 — STATIC ANALYSIS

Config audit

32 checks across gateway settings, file permissions, channel policies, tool restrictions, and credential storage. Finds misconfigured defaults before attackers do.

RING 2 — LIVE PROBES

Behavioral verification

Actually connects to your running gateway. Is the port open on non-loopback interfaces? Does it require auth? Does /health leak secrets? Config could lie. Probes can't.

RING 3 — SUPPLY CHAIN

Skill scanner

Scans every installed skill — not just SKILL.md, but all .js, .sh, .py scripts — for credential theft, RCE, exfiltration, and prompt injection patterns.

Get started

Up and running in 30 seconds

npm package. Zero dependencies. Works on macOS, Linux, Windows (WSL).

$clawarmor audit# run your first audit
$clawarmor fix --apply# auto-fix safe issues
$clawarmor scan# scan installed skills

Requires Node.js 18+ and an OpenClaw installation.
